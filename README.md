# \# Apple Inc. (AAPL) – Integrated Financial Model \& DCF Valuation



This repository contains a fully integrated three-statement financial model and Discounted Cash Flow (DCF) valuation of Apple Inc. (NASDAQ: AAPL).



The objective of this project was to build an institutional-style valuation model from scratch by forecasting Apple's operating performance, linking the Income Statement, Balance Sheet, and Cash Flow Statement, and deriving an intrinsic equity value using the Free Cash Flow to Firm (FCFF) methodology.



\## Model Highlights



\### Financial Statement Modeling



\* Fully linked Income Statement, Balance Sheet, and Cash Flow Statement

\* Historical financial analysis and forecast period projections

\* Dynamic operating driver assumptions

\* Revenue forecasting by business segment

\* Margin and expense forecasting framework

\* Working capital forecasting schedule

\* PP\&E roll-forward schedule

\* Equity roll-forward schedule



\### Valuation Framework



\* Unlevered Free Cash Flow (FCFF) valuation

\* Weighted Average Cost of Capital (WACC) calculation

\* Terminal Value using the Gordon Growth Method

\* Enterprise Value to Equity Value bridge

\* Intrinsic value per share calculation

\* Sensitivity analysis across WACC and Terminal Growth assumptions



\### Model Integrity Checks



\* Balance Sheet balancing checks

\* Cash flow reconciliation checks

\* PP\&E schedule reconciliation

\* Equity schedule reconciliation

\* Circular reference review
